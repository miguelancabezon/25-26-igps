Paso 1:Clonamos el repositorio remoto dentro de nuestro ordenador para poder trabajar dentro de él.

Paso 2:Creamos una carpeta con nuestro nombre dentro de la carpeta de actividades.

Paso 3:Dentro de la carpeta con nuestro nombre creamos otra con el nombre AEC-GIT.

Paso 4:Dentro de ella creamos un documento de texto.

Paso 5: Al ejecutar el comando git status podemos ver que hay cambios sin trackear.

Paso 6: Ejecutamos el comando git add * para añadir todos los cambios a git.

Paso 7: Ejecutamos el comando git commit -m "nombre del commit" para guardar todos los cambios. Al ejecutar el comando git status podemos ver que todo esta en orden de nuevo.

Paso 8: Podemos hacer cambios dentro de el documento de texto para ver que ocurre.

Paso 9: Si volvemos a ejecutar el comando git status podemos comprobar que vuelve a haber cambios sin commitear.

Paso 10: Volvemos a ejecutar los comandos git add * y git commit -m "nombre del commit" para dejar todo guardado.

Paso 11: Para terminar debemos crear una rama nueva, para ello usamos el comando git checkout -b "docs/modificaciones"

Paso 12: Modificamos el documento de texto con el que comenzamos para explicar todos los cambios realizados. Además en la misma carpeta añadimos capturas con los cambios realizados. Para terminar subimos todo al repositorio remoto con git push.
