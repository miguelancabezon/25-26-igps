## Diagramas a diseñar 1 - Mindmap

El primer diagrama, el mindmap, va a servir cómo ordenación de las ideas que se te van a ir ocurriendo. Aquí deberás separar los distintos aspectos que puede tener el juego.

*Requisitos*

- Usar estilos personalizados
- Al menos 5 colores diferentes
- Mínimo 3 niveles de profundidad
- Incluir emojis o símbolos
- Usar ambos lados del mindmap

## Diagramas a diseñar 2 - Gantt

Con el diagrama de Gantt, vamos a proponer una línea de tiempo dónde se diseñe, desarrolle, pruebe y lance el videojuego.
Para estandarizar, el diagrama deberá tener las siguinetes fases:
1. Fase 1 - Preproducción
2. Fase 2 - Producción
3. Fase 3 - Postproducción
4. Hitos

Además deberá tener los siguientes hitos
- Milestone 1: Prototipo
- Milestone 2: beta
- Milestone 3: v1.0

*Requisitos*

- Al menos 15 tareas
- Usar colores por categoría
- Incluir dependencias
- Agregar notas con recursos/equipo
- Porcentajes de completado
