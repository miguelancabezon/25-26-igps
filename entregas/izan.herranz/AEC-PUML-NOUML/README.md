# AEC-PUML-NOUML — Survival

Este repositorio incluye cuatro diagramas PlantUML (NoUML) para un videojuego de supervivencia:

1. **Mindmap** (`mindmap-survival.puml`): ideas principales del juego.  
2. **Gantt** (`gantt-survival.puml`): planificación por fases con hitos.  
3. **JSON** (`json-model-survival.puml`): modelado de objetos/clases del juego, acciones y metadatos.  
4. **Wireframe/Salt** (`wireframe-survival.puml`): boceto de interfaz con tabs, tabla e interacciones.
