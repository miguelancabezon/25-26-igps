#  Diagrama de Gantt — Survival Game

##  Fases del proyecto

1. **Preproducción**
   - Documento de diseño (GDD)
   - Concept art 
   - Plan técnico y planificación
   

2. **Producción**
   - Sistema de crafteo 
   - Sistema de IA 
   - Mundo inicial (bosque) 
   - HUD / UI 
   - Sonido y ambiente 
   - Servidor multijugador 
   

3. **Postproducción**
   - Optimización y corrección de bugs 
   - Localización (EN/ES)
   - Marketing y tráiler
