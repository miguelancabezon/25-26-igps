# Human Evolution Game - AEC-PUML-NOUML

Proyecto para la Actividad de Evaluación Continua. El juego simula la evolución humana desde la Edad de Piedra hasta la era moderna, enfrentando retos y evolucionando como sociedad. Puedes competir y colaborar con otros países en la misma fase evolutiva.

## Estructura de la carpeta

- **mindmap/**: Diagrama de ideas (PlantUML)
- **gantt/**: Cronograma de desarrollo (PlantUML)
- **json/**: Definición de objetos y clases (JSON)
- **wireframe/**: Boceto de interfaz (PlantUML Salt)
- **README.md**: Explicación y guía del proyecto
